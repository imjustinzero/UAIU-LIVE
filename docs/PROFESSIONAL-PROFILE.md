# Professional Profile System

## What it is
Professional profiles provide a persistent public record of verified contributors (partners and ISO verifiers), similar to a carbon-market ORCID profile.

## ORCID integration
- Add ORCID ID to profile settings.
- Display ORCID publicly on `/x/professionals/:id`.

## Linking publications
Attach DOI-linked publication arrays to the profile to strengthen attribution and scholarship traceability.

## Contribution tracking
Profiles aggregate:
- Methodologies authored
- Verification volume and geography
- Working group memberships
- Standards contributions
- Peer review contributions (MQL)
- Reputation score

## External usage
Profiles are designed to be shareable for conferences, procurement due diligence, CV/academic references, and standards body participation.
