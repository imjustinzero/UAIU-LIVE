# UAIU.LIVE/X MRV Methodology Reference

## GHG Protocol
Baseline emissions and reductions are converted to CO2e using standard emission factors from methodology configuration and sensor-derived activity data.

## Verra VM0015 (REDD+)
Forest protection impact is modeled from measured/semi-observed biomass/cover proxies with conservative avoidance factor controls.

## Gold Standard TPDDTEC (biogas)
Methane destruction and fossil displacement contributions are converted into CO2e-equivalent issuance quantities.

## Data quality requirements
Credit issuance quality considers:
- Signature validity ratio
- Coverage completeness over reporting period
- Unresolved anomaly burden
- Sensor continuity/offline gaps
- Satellite consistency checks when available

## Audit chain embedding in MRV reports
Each MRV report links to the underlying immutable chain blocks for:
- Sensor measurements
- Anomaly detection events
- MRV report creation
- Credit issuance actions

PDF exports include first/last block references across the reporting interval.
