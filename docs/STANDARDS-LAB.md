# UAIU Standards Lab

## What it is
UAIU Standards Lab is an open methodology development environment where working groups form, draft, comment, ballot, and publish standards with immutable records.

## Forming a working group
1. Create a working group with name, lead organization, and target standard.
2. Invite members by platform ID or email.
3. Set lifecycle stage: forming → active → consultation → balloting → published.

## Draft development flow
- Publish drafts (WD/CD/FDIS labels supported).
- Drafts are rendered with section anchors and paragraph-level comment targeting.
- New drafts include diff metadata from the prior draft.

## Comments and voting
- Anyone can read drafts.
- Verified partners can comment.
- Verified partners and ISO verifiers can vote.
- Votes are tallied by ballot and exposed publicly after close.

## Audit chain recording
Draft publications, comments, and votes are recorded with audit block references so standards governance is transparent and tamper-evident.

## For ISO bodies and standard developers
Standards Lab supports consultation rounds, formal disposition tracking, and publication-ready transition from draft to methodology release.
