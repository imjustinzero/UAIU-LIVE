# UVS Public Verification Guide

1. Scan the certificate QR code or open `/x/verify/:certificateNumber`.
2. Compare certificate hash shown online vs PDF hash.
3. Check audit chain block references.
4. Download evidence package ZIP.
5. Recompute evidence hash locally and compare.
6. Confirm verifier statement and registry proof references.

This process enables independent verification without trusting UAIU-hosted claims.
