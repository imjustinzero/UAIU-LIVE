# Professional Profile Guide

## Creating and claiming your profile
- Create/update via `POST /api/professionals/profile`.
- Required fields: `displayName`, `organization`.
- Public URL pattern: `uaiu.live/x/professionals/{id}`.

## Linking ORCID iD
- Endpoint: `POST /api/professionals/profile/orcid`.
- Format required: `0000-0000-0000-0000`.

## Adding publications via DOI
- Add DOI links in your profile/methodology records.
- These contribute to academic references in impact and MQI calculations.

## How methodology authorship appears
Authored methodologies are joined from `partner_methodologies` and displayed in public profile output.

## Verification track record calculation
Profile includes verification history derived from `verification_statements` (count + tonnes).

## Using your profile externally
- Add profile URL to grant proposals, verifier submissions, and procurement dossiers.
- Share APA/BibTeX/RIS citations generated by impact endpoint.

## Citation format examples
- APA: `Author. (Year). Title. UAIU Methodology Library. URL`
- BibTeX:
```bibtex
@misc{uaiu_methodology_2026,
  author = {Author Name},
  title = {Methodology Title},
  year = {2026},
  howpublished = {UAIU Methodology Library},
  url = {https://uaiu.live/x/methodologies/...}
}
```
- RIS:
```ris
TY  - GEN
TI  - Methodology Title
AU  - Author Name
PY  - 2026
UR  - https://uaiu.live/x/methodologies/...
ER  -
```

## Impact metrics calculation
Uses credits, tonnes CO2e, countries, standards, and citation counts linked to methodologies authored by the professional.
