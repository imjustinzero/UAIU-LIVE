# Monitoring and Alerts

## Add alerts for
- server errors
- failed webhooks
- failed signins and lockouts
- Stripe failures
- slow routes > 1500ms
- failed email sends
- restart count / uptime drops
- memory > threshold

## Suggested vendors
- Sentry for errors
- Better Stack / UptimeRobot for uptime
- Postgres backup monitor
- Slack / email notification channel
